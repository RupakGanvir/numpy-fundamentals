# NumPy Fundamentals

A beginner-friendly NumPy learning repository covering the essential concepts required for Data Analysis, Machine Learning, and Artificial Intelligence.

---

## Topics Covered

- Creating Arrays
- Array Attributes (`shape`, `dtype`, `ndim`, `size`, etc.)
- Indexing and Slicing
- Working with 2D and 3D Arrays
- Array Initialization
- Mathematical Operations
- Universal Functions (UFuncs)
- Linear Algebra Basics
- Statistical Operations
- Reshaping Arrays
- Flatten vs Ravel
- Broadcasting
- Transpose
- Boolean Masking
- Advanced Indexing
- Useful NumPy Functions
- Practice Exercises
- Key Takeaways

---

## Learning Outcomes

After completing this notebook, you will be able to:

- Create and manipulate NumPy arrays efficiently.
- Understand multidimensional arrays and indexing techniques.
- Perform vectorized mathematical operations.
- Use broadcasting for operations on arrays with different shapes.
- Filter data using boolean masking.
- Reshape, transpose, flatten, and reorganize arrays.
- Apply basic linear algebra and statistical operations.
- Build a strong NumPy foundation for Pandas, Scikit-learn, and Deep Learning.
